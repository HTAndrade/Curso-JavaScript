O que é JavaScript?

.É uma linguagem de programação de alto nivel - "User side";

.Originalmente chamada de LiveScript;

.Porém recebeu nome de JavaScript por causa da grande fama de Java;

.JavaScript = JS = Vanilla JavaScript = ECMAScript;

.Criada para deixar as páginas web vivas;

.Just-in-time compiled;

.Orientada a objetos;

.Criada em 1995;
